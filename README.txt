=== OVERVIEW ===

This code accompanies a talk I gave at edUi 2011 ("Advanced WordPress
Features & Customizations") on October 14 in Richmond, Virginia. For
more information, visit the following links.

* http://randyhoyt.com/wordpress/advanced-customizations/


=== INSTALLATION ===

1. Download and unzip the source code.
2. Upload the plugins directory and themes directory into
   the wp-content directory in WordPress.
3. Activate the Custom Functionality Plugin.
4. Activate the EMAC 3361 theme.  


=== REQUIREMENTS ===

* WordPress 3.2.1 with the TwentyEleven theme installed.
